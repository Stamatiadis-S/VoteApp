  • Request
  • Response

Headers

  • Content-Type: required (string)
  • Accept: required (string)

Body

Media type: application/json

Type: object

Properties
  • data: required (object)
    • type: required (string - default: users)
    • attributes: required (object)
      • email: required (string)
      • username: required (string - minLength: 5 - maxLength: 20 - pattern: ^[A-Za-z_][A-Za-z0-9_]{5,20}$)
      • password: required (string - minLength: 8 - maxLength: 20)

        Password must contain at least one lowercase, one uppercase, one digit and one of the following symbols (!@#$%^&+=).

Example:

{
  "data": {
    "type": "users",
    "attributes": {
      "email": "a@a.com",
      "username": "account1",
      "password": "accounT1!"
    }
  }
}

        Single or Multiple option voting.

        Poll subject.

      • options: required (array of items)

        Poll available options. There is no blank option by default.

        Items: items

        • option: required (string - minLength: 1 - maxLength: 100)

{
  "data": {
    "type": "polls",
    "attributes": {
      "visibility": "Private",
